# My Portfolio Website

A modern and responsive personal portfolio website built using HTML and CSS.
This website showcases my profile, skills, and contact information in a clean and interactive user interface.

---

## Features

* Responsive portfolio layout
* Sticky navigation bar
* Smooth scrolling navigation
* Hero landing section
* About Me section
* Skills showcase
* Contact form
* Social media links
* Modern dark UI design
* Hover effects and animations

---

## Technologies Used

* HTML5
* CSS

---

## Website Sections

### Home

Introduction section with welcome message and quick navigation.

### About Me

Brief introduction about me along with highlights and dedication.

### Skills

Displays my technical skills using icons and cards.

### Contact

Includes contact form, email, phone number, and social media links.

* Dark modern theme

* Clean typography
* Responsive flexbox layout
